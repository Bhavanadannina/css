CSS Box Model Demonstration

Content Box Model (Default)
This box uses the content-box model.

In the content-box model, the width and height of the box are applied to the content only. Padding and border are added outside the content area.

Border Box Model
This box uses the border-box model.

In the border-box model, the width and height of the box include the content, padding, and border. The total width and height of the box do not change with the addition of padding and border.

Effects of Padding, Border, and Margin

Padding
This box has increased padding (40px).

Padding increases the space inside the border, pushing the content inward.

Border
This box has a thicker border (10px).

The border surrounds the padding and content, increasing the overall size of the box.

Margin
This box has increased margin (40px).

Margin increases the space outside the border, pushing the box away from other elements.
